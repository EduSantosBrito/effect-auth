# Security Policy

## Reporting a Vulnerability

Please do not open public issues for suspected vulnerabilities.

Report vulnerabilities privately through GitHub security advisories for this repository. Include:

- Description of the vulnerability.
- Steps to reproduce.
- Potential impact.
- Suggested mitigation, if known.
- Affected versions, if known.

## Disclosure

If the issue is confirmed, we will release a patch as soon as possible and disclose after a fix is available.

## Supported Versions

Only the latest published version is supported.
